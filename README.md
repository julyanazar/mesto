# Проект: Место

## Учебный проект по изучению базы js и работы с браузером
**В проекте используются следующие технологии:**
* flexbox-вёрстка;
* Grid Layout
* позиционирование элементов;
* методология БЭМ;
* файловая структура и файловые пусти (по БЭМ);
* адаптивная резиновая верстка;
* JavaScript (DOM).
